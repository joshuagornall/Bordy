## Get Started (Unleash Your Inner Border Artist)

1. **Clone the Repository (Summon Bordy)**
   ```bash
   git clone https://github.com/joshuagornall/bordy.git
   ```
2. **Enter the Bordy Realm**
   ```bash
   cd bordy
   ```
3. **Awaken the Dependencies**
   ```bash
   npm install
   ```
4. **Bring Bordy to Life**
   ```bash
   npm start
   ```
